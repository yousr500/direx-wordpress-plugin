# Direx – WordPress Authentication Plugin

**Direx** is a secure and customizable WordPress plugin developed as part of a web development internship project. It provides a role-based authentication system and basic order management functionalities for WordPress-based applications.

## 🔐 Features

- Role-based login system (Admin/User)
- Encrypted session handling
- Order creation and management
- Dashboard UI for user interaction
- Cross-browser compatibility

## 🛠️ Built With

- PHP
- JavaScript (ES6)
- HTML5 / SCSS
- WordPress Plugin API
